How it started?

I wanted a static blog. Most static site generators are either a bit bloat or lack some features.

So I decided to make a minimal but feature rich SSG from scratch. There were plenty of powerful programming languages to choose from like :

  • Python
  • Ruby
  • JavaScript

So I decided to use the one which will make me suffer the most. That's right.. BASH!

Why bash?

IDK, maybe because I like pain and suffering...
